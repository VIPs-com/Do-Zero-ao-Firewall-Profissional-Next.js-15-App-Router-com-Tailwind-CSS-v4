## 🎯 Sobre o Projeto

Este workshop foi projetado para levar **iniciantes a profissionais** através de uma experiência imersiva:

- Laboratório real com Ubuntu Server
- Explicações camada por camada do **Modelo OSI**
- Checklists interativos + sistema de badges
- Quiz técnico com 26 questões
- Certificado digital com código de autenticidade
- Topologia interativa da rede

**Objetivo:** Você não só lê — você **constrói, valida e conquista** cada etapa.

---

## ✨ Principais Funcionalidades

- **Gamificação completa** — Badges, progresso visual e notificações
- **Quiz + Certificado** — Só libera com desempenho mínimo
- **Busca global** — Pressione `Ctrl + K`
- **Mergulhos Profundos** — Conteúdo avançado em modais
- **Modo Dark/Light** — Experiência confortável
- **Totalmente responsivo** — Funciona em desktop e tablet

## ⚙️ Como Rodar o Projeto

### Pré-requisitos

*   Node.js 20+
*   Git

### Passo a Passo

```sh
# 1. Clone o repositório
git clone https://github.com/SEU_USUARIO/workshop-linux-firewall.git
cd workshop-linux-firewall

# 2. Instale as dependências
npm install

# 3. Rode em desenvolvimento
npm run dev
```

Acesse → [http://localhost:3000](http://localhost:3000)
